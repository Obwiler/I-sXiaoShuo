# 新对话窗口承接协议（Session Handoff）

> 当你打开一个新 Codex 对话窗口继续写小说时，不需要重新解释"我在写什么"。把这份文件扔给新会话，它就能无缝接上。

## 一、核心文件：`项目承接.md`

位于项目根目录 `《书名》/项目承接.md`。这是新会话的"一键恢复包"。

### 格式（固定结构，新会话逐段读取）

```markdown
# 《书名》项目承接 · 请先读完本文件再操作

## 快速定位
- 平台：番茄 / 起点 / 晋江
- 频道：男频 / 女频
- 题材：XXX
- 当前字数：约XX万字
- 当前章节：第XX章 / 第X卷
- 签约状态：未签约 / 已签约 / 首秀中

## 这次要做什么
一句话说清楚这次打开 Codex 的目的：
- [ ] 继续写第XX章
- [ ] 修改前三章
- [ ] 审稿/体检
- [ ] 准备投稿
- [ ] 其他：___

## 关键文件清单（Codex请按需打开）
- 记忆锚点：`记忆锚点.md` ← 先读这个，全书状态全在里面
- 设定总纲：`设定/00_设定总纲.docx`
- 人物档案：`设定/01_人物档案.docx`
- 大纲章纲：`设定/02_大纲章纲.docx`
- 当前正文：`正文/第XX章.docx`
- 已发布章节：`已发布/` （参考已发内容的一致性）
- 进度日志：`工程/进度日志.md`

## 上次会话做了什么
（填上次会话结束时的操作，3句话以内）

## 当前阻塞/待解决问题
- 
- 

## 特殊提醒
- 这次要注意什么：
- 上次容易犯的错：
- 用户最近强调过：
```

## 二、新会话启动标准流程

Codex 在新会话中读到"写小说""继续写""第X章"等关键词时，自动执行：

```
1. 搜索工作区下所有 `项目承接.md` 文件
2. 如果没有 → 提示用户提供承接文件或创建新项目
3. 如果有 → 按以下顺序读取：
   a. 项目承接.md（了解任务）
   b. 记忆锚点.md（恢复全书状态）
   c. 按需读设定/大纲/上章正文（执行任务）
```

## 三、与 Codex Memory 的配合

| 存什么 | 存在哪 | 新会话怎么读 |
|---|---|---|
| 全书压缩快照 | `记忆锚点.md` | 承接流程自动读 |
| 本次任务说明 | `项目承接.md` | 承接流程自动读 |
| 用户偏好/决策记录 | Codex memories/ | Codex 自动加载 |
| 详细设定/大纲/正文 | 对应目录下的文件 | 按需手动打开 |

## 四、多书切换

如果同时写多本书，每本书有自己的 `项目承接.md`：

```
E:\ZhuoMian\
├── 奥贝维勒\
│   └── 项目承接.md          ← 指向第一本书
├── 第二本书\
│   └── 项目承接.md          ← 指向第二本书
```

新会话时告诉 Codex "打开奥贝维勒的项目承接"，它就会定位到正确的书。

## 五、会话结束 Checklist

每轮 Codex 会话结束前，Codex 自动执行：

- [ ] 更新 `记忆锚点.md` 的所有七段
- [ ] 更新 `项目承接.md` 的"上次会话做了什么"和"当前阻塞"
- [ ] 更新 `工程/进度日志.md`
- [ ] 确认正文文件已保存
- [ ] 如有投稿/签约变动，更新投稿记录

## 六、跨团队交接

当将项目移交给另一个团队或个人时，除本文件外还需要：

### 必须移交的文件

| 文件 | 用途 | 是否必需 |
|------|------|---------|
| 项目承接.md | 项目状态和下一步任务 | 必需 |
| 记忆锚点.md | 全书七段快照 | 必需 |
| AI注意事项及规则/ | 项目专属写作规则 | 必需 |
| 设定/ | 世界观/人物/大纲/伏笔 | 必需 |
| 工程/ | 进度日志/投稿记录 | 强烈建议 |
| PROJECT.yaml | 工程元数据（如使用） | 可选 |
| 报告/ | 修整历史记录 | 可选 |

### 接收方启动流程

接收方在新环境中执行：
1. 阅读 skill 的 skill-handoff.md 了解 skill 整体结构
2. 阅读本项目的 项目承接.md 了解任务
3. 阅读 记忆锚点.md 恢复全书状态
4. 阅读 AI注意事项及规则/05_快速接入指南.md 了解文件路由
5. 按需打开 设定/ 下的详细文档
6. 开始工作

### 交接验证

接收方完成设置后，应能：
- 找到 项目承接.md 和 记忆锚点.md
- 理解当前写到哪一步
- 找到模板和规则文件
- 启动一次写作流程而不需要向外追问
