# 大纲情节分析 · Outline Analysis
`
> 定义对大纲/章纲进行系统性情节分析的方法论。适用于模式C（项目工程化模式）。
`
---
`
## 分析维度（六项指标）
`
### 1. 因果链完整性
检查每个情节事件是否有明确的前因后果。
标准：每个事件前必有触发，后必有后果。不能出现角色突然决定或无后续影响。
判定：通过>=90%, 警告80-89%, 失败<80%
`
### 2. 信息密度
检查每节大纲的事件节点数量。
节点定义：行动、信息发现、决策、环境变化。
对照表：2000-2500字=5-6节点, 2500-3500=7-8, 3500-5000=10-12
判定：通过>=推荐下限, 警告=达最低, 失败<最低
`
### 3. 弧段张力曲线
检查全书张力分布。参考龙族三节奏：三章一波浪。
标准：每个弧段至少一个波峰，弧间过渡不陡降，弧段三为最高峰，每段尾有收束。
`
### 4. 角色动机链
检查核心角色动机一致性。
标准：每个角色有明确动机，动机变化有触发，优先级因压力合理变化，行为与理性分一致。
`
### 5. 伏笔-回收匹配
检查伏笔是否有回收位置。
标准：标注回收弧段，同卷最长3弧段，跨卷标记，不能无回收。
`
### 6. 五要素覆盖
检查每节是否包含完整五要素：前因、过程、后果、动机、影响。
判定：通过>=90%, 警告80-89%, 失败<80%
`

## 弧段自定义（命令行参数）
`
使用 --arcs 参数可以自定义弧段划分。
`
### 基本用法
`
`
# 默认（使用大纲自带弧段标签）
python scripts/analyze_outline.py
`
# 自定义弧段范围
python scripts/analyze_outline.py --arcs "起航(001-007);探索(008-019);决战(020-033);余波(034-043);终局(044-052)"
`
`
### 格式说明
`
弧段定义格式：名称(起始章-结束章);名称(起始章-结束章)
章节编号支持三位数或普通数字。
弧段定义按顺序解析，不支持嵌套。
`
### 分项检查
`
`
python scripts/analyze_outline.py --check "1,3"
python scripts/analyze_outline.py --check "all"
`
`
可选项：1=五要素覆盖, 2=因果链完整性, 3=弧段结构, 4=角色动机覆盖, 5=伏笔标记
`
### JSON输出
`
`
python scripts/analyze_outline.py --json
