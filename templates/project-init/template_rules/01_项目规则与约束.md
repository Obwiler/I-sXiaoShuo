## 二、叙事规则

1. 每章必须回答四件事：主角想做什么 / 谁阻止他 / 局面变成什么 / 读者为什么往下看
2. 每场戏必须有：目标 → 阻碍 → 变化
3. 信息揭示通过角色视角，不写全知叙述者
4. 章末必须有钩子——在「即将发生什么」的前一秒断章

---

## 三、番茄平台约束

| 位置 | 必须完成的事 |
|------|-------------|
| 前3句 | 立危机或反差 |
| 前1000字 | 第一次小爽点（能力展示/打脸/反转三选一） |
| 第1章末 | 金手指/核心卖点上线 + 章末钩子 |
| 第3章末 | 第一个小高潮 + 主线悬念清晰 |

## 四、敏感避规

- 政治影射用架空世界锚定
- 冲突在个体之间，不在阶级之间
- 不写群体性事件
